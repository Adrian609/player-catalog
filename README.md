# Player Catalog

> **Demo:** <https://adrian609.github.io/player-catalog/>  
> **View repo in VSCode:**
 <https://github1s.com/Adrian609/player-catalog/>

 **App supports querying or sorting by:**

- division
- year
- team

**The attributes available for players:**

- playerID
- yearID
- stint
- teamID
- lgID
- G
- AB
- R
- H
- 2
- B
- 3B
- HR
- RBI
- SB
- CS
- BB
- SO
- IBB
- HBP
- SH
- SF
- GIDP
